# Won-t-you-be-my-K-nearest-neighbor-
This report analyzes the MNIST dataset (60,000 28×28 pixel images of digits 0–9) using sklearn implementations of KNN and KMeans. KNN is a supervised algorithm that classifies data based on the k nearest neighbors. KMeans is an unsupervised clustering method that iteratively optimizes k centroids based on the mean of data clusters.
